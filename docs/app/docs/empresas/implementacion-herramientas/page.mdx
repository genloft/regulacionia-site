# Implementación y herramientas prácticas

## Módulos

### Módulo 1: Plataformas low-code
Uso de herramientas como Google Cloud AI o Microsoft Azure que permiten implementar soluciones sin necesidad de programación avanzada.

### Módulo 2: Integración en CRM y ERP
Cómo conectar sistemas de IA con tu software de gestión actual para enriquecer los datos de clientes y operaciones.

### Módulo 3: Automatización de procesos (RPA)
Uso de UiPath y otras herramientas para automatizar tareas repetitivas administrativas.

## Taller
*   **Taller:** Crea un chatbot simple para atención al cliente.
*   **Caso práctico:** Optimización de una cadena de suministro utilizando datos predictivos.
