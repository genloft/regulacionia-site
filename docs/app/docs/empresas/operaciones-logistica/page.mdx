# IA en operaciones y logística

## Módulos

### Módulo 1: Predicción de demanda
Algoritmos para anticipar picos de venta y optimizar el inventario.

### Módulo 2: Optimización de rutas
Caso de estudio de UPS y cómo optimizan millones de rutas de entrega diariamente.

### Módulo 3: Mantenimiento predictivo
Uso de sensores e IA para predecir fallos en maquinaria antes de que ocurran.
