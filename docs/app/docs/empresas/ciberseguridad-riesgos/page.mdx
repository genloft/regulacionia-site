# Ciberseguridad y riesgos empresariales

## Módulos

### Módulo 1: IA para detección de fraudes
Sistemas que analizan transacciones en tiempo real para bloquear actividades sospechosas.

### Módulo 2: Ataques IA (Adversarial ML)
Entender cómo los atacantes pueden engañar a los modelos de IA y cómo protegerse.

### Módulo 3: Marcos de compliance
Introducción a la norma ISO 42001 sobre sistemas de gestión de inteligencia artificial.

## Proyecto
*   **Proyecto:** Diseña un plan básico de ciberseguridad que contemple riesgos específicos de IA.
