# Sostenibilidad ambiental

## Módulos

### Módulo 1: Consumo energético
El coste oculto de entrenar grandes modelos de lenguaje (LLMs) como GPT-3 o GPT-4.

### Módulo 2: IA verde
Técnicas para optimizar el código y el hardware y reducir la huella de carbono de la IA.

Cómo la IA puede ayudar a predecir desastres climáticos y optimizar redes de energía renovable.
