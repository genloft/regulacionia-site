# IA en marketing y ventas

## Módulos

### Módulo 1: Personalización de campañas
Cómo la IA permite segmentar audiencias en tiempo real y ofrecer mensajes personalizados a escala.

### Módulo 2: Predicción de churn y leads
Uso de modelos predictivos para identificar clientes en riesgo de abandono y priorizar oportunidades de venta.

### Módulo 3: Análisis de sentimientos
Monitoreo de redes sociales para entender la percepción de marca mediante análisis de lenguaje natural.

## Actividades
*   **Video:** Análisis de campañas reales (ej. Coca-Cola con IA Generativa).
*   **Actividad:** Segmenta una base de datos de prueba utilizando herramientas de IA.
