# Gestión de talento y RRHH con IA

## Módulos

### Módulo 1: Reclutamiento automatizado
Herramientas para filtrar CVs y encontrar el talento adecuado más rápido, reduciendo sesgos (si se usan correctamente).

### Módulo 2: Evaluación de desempeño
Nuevas métricas y análisis de productividad basados en datos.

### Módulo 3: Bienestar empleado
Análisis de patrones de trabajo para prevenir el burnout y mejorar la satisfacción laboral.

## Evaluación
*   **Evaluación:** Realiza una auditoría simulada de los procesos de RRHH de tu empresa para identificar áreas de mejora con IA.
